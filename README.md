## dotnet command

- dotnet new -l

- dotnet tool list -g

## dotnet ef command

- dotnet tool install --global dotnet-ef --version 5.0.12
- dotnet ef
- dotnet ef migrations add InitialMigrations -o Data/Migrations
- dotnet ef database drop
- dotnet ef database update
